Detailed Description

This is the documentation for Adafruit's ADG72x driver for the Arduino platform. It is designed specifically to work with the ADG728 and ADG729 analog multiplexers.

These devices use I2C to communicate, and require 2 pins (SCL+SDA) Two more pins can be used for the optional address configuration, allowing multiple devices to share the same I2C bus.
